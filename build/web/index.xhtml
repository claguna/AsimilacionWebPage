Asimilacion













Este proyecto se enmarca dentro de un convenio entre el Instituto de Matemáticas y el Instituto de Ingeniería de la UNAM. El objetivo del mismo es la integración de un proceso de asimilación de datos sobre los modelos de predicción del Instituto de Ingeniería para los niveles de los ríos en la zona de Villahermosa, Tabasco. La finalidad última de este trabajo consiste en establecer un mecanismo de protección y alarma para posibles inundaciones en la zona de Villahermosa.
